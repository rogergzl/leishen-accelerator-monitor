# 雷神加速器 · 时长监控助手

> 再也不会忘记暂停加速时长了！

一个轻量级的 Windows 后台监控工具，专门解决**雷神加速器退出后忘记暂停时长**导致时长空跑的痛点。

---

## 功能

| 功能 | 说明 |
|------|------|
| 加速器退出提醒 | 检测到雷神加速器进程退出时，弹窗提醒暂停时长 |
| 全屏程序退出提醒 | 全屏运行时不打扰，仅在退出全屏后才弹窗提醒检查加速时长 |
| 关机拦截 | 关机时如果加速器还在后台运行，弹出确认框阻止关机（每次关机都会检查，只要加速器在运行就会拦截） |
| 事件驱动 | 基于 WMI + WinEvent 内核事件，不轮询、不占 CPU、不脏日志 |
| 图形化管理 | 双击即出管理界面，一键安装 / 停止 / 卸载 |

### 行为说明

- **加速器退出提醒**：加速器进程从运行变为不运行时，弹出一个警告框提醒你暂停时长
- **全屏退出提醒**：在游戏等全屏程序运行期间**不会弹任何窗口**，只有当你退出全屏回到桌面后，才会弹窗提醒检查加速时长（有 5 秒冷却，不会重复弹）
- **关机拦截**：Windows 关机流程中，如果加速器仍在运行，会弹出确认框。点"是"取消关机（去暂停时长），点"否"放行关机。这是**每次关机都会触发**的检查——只要加速器进程还在，无论取消多少次关机，下次关机依然会拦截。只有加速器退出后再关机才不会弹窗。

## 技术细节

```
技术栈: Python 3 + tkinter + WMI + WinEvent + Win32 API
打包:   PyInstaller --onefile --windowed
大小:   ~13MB（tkinter 内置于 Python 标准库，无需额外运行时）
兼容:   Windows 7 / 8 / 10 / 11
```

### 监控原理

**加速器退出**：通过 WMI 订阅 `Win32_ProcessStopTrace` 事件，内核在目标进程退出时主动通知，无需轮询进程列表。

**全屏检测**：通过 `SetWinEventHook` 监听前台窗口变化和窗口销毁事件。全屏运行期间不做任何打扰；仅当检测到全屏窗口关闭或前台从全屏切换到非全屏窗口时，触发提醒。

**关机拦截**：子类化窗口的 `WM_QUERYENDSESSION` 消息，在 Windows 询问"是否可以关机"时检查加速器状态。若加速器运行中，返回 FALSE 取消本次关机，同时用 `ShutdownBlockReasonCreate` 在关机画面上显示原因文字。

### 开发

```bash
# 运行管理界面
python leishen_monitor.pyw

# 后台 daemon 模式（由计划任务调用）
python leishen_monitor.pyw --daemon

# 打包为单文件 exe
pip install pyinstaller
pyinstaller --onefile --windowed --name "雷神时长助手" leishen_monitor.pyw
```

### 自定义监控进程

编辑 `leishen_monitor.pyw` 顶部的 `PROCESS_NAMES` 列表：

```python
PROCESS_NAMES = [
    "leigod.exe",              # 雷神加速器
    "your_game_booster.exe",   # 你的其他加速器
]
```
